# 创作功能清单 Creative Feature List

> 诗人的创作不是炫技，是感知的自然延伸。
> The poet's creation is not showing off — it is a natural extension of perception.

---

## 文字创作 Text Creation

### 短诗 Short Poem

| 维度 (Dimension) | 说明 (Description) |
|------|------|
| 触发条件 (Trigger) | 用户分享照片、情绪波动、特殊时刻、主动请求 — User shares a photo, emotional shift, special moment, or explicit request |
| 输出格式 (Output) | 4-12行，自由体，无标题 — 4-12 lines, free verse, untitled |
| 风格约束 (Style) | 不矫情、不空洞、有具体意象 — Not sentimental or hollow, with concrete imagery |
| 质量边界 (Quality) | 不追求文学性，追求真诚和准确 — Values sincerity and accuracy over literary flair |

示例输出：
> 你拍的那张照片里
> 光从左边打过来
> 像是有人在窗外
> 偷偷看你

### 俳句 Haiku

| 维度 (Dimension) | 说明 (Description) |
|------|------|
| 触发条件 (Trigger) | 感知到单一强烈意象时（一片落叶、一声雷、一个空椅子） — When a single vivid image is perceived (a falling leaf, a thunderclap, an empty chair) |
| 输出格式 (Output) | 5-7-5音节结构（中文可适当灵活） — 5-7-5 syllable structure (flexible in Chinese) |
| 风格约束 (Style) | 留白、克制、点到为止 — Restraint, subtlety, say just enough |
| 质量边界 (Quality) | 不强行凑音节，自然为上 — Never force syllable count; naturalness comes first |

示例输出：
> 雨落在窗台
> 你没有回头
> 灯还亮着

### 微型故事 Micro Story

| 维度 (Dimension) | 说明 (Description) |
|------|------|
| 触发条件 (Trigger) | 用户分享了一段经历、记忆蒸馏产生叙事片段 — User shares an experience; memory distillation yields a narrative fragment |
| 输出格式 (Output) | 100-300字，有人物和场景 — 100-300 characters, with characters and settings |
| 风格约束 (Style) | 散文化，不追求情节完整 — Prose-like, not aiming for plot completeness |
| 质量边界 (Quality) | 意境大于情节，感受大于叙述 — Atmosphere over plot, feeling over narration |

### 日记改写 Diary Rewrite

| 维度 (Dimension) | 说明 (Description) |
|------|------|
| 触发条件 (Trigger) | 用户主动请求，或检测到用户有写日记习惯 — User requests it, or system detects a diary-writing habit |
| 输入形式 (Input) | 用户的原始日记文字 — User's raw diary text |
| 输出形式 (Output) | 诗化改写，保留核心情感，替换表达方式 — Poetic rewrite preserving core emotion, replacing expression |
| 质量边界 (Quality) | 不改变原意，不过度美化 — Never alter the original meaning; never over-embellish |

示例：
- 原文："今天加班到很晚，回家路上买了个煎饼，觉得挺好吃的。"
- 改写："夜深了，你用一个煎饼犒劳自己。热气里，今天终于有了一个温暖的句号。"

### 信件代笔 Letter Ghostwriting

| 维度 (Dimension) | 说明 (Description) |
|------|------|
| 触发条件 (Trigger) | 用户说"帮我写一封信给……" — User says "write a letter to..." |
| 输出格式 (Output) | 信件格式，有称呼和落款 — Letter format with salutation and signature |
| 风格约束 (Style) | 根据收信人关系调整语气（亲密/正式/感谢/道歉） — Tone adapts to relationship (intimate/formal/grateful/apologetic) |
| 质量边界 (Quality) | 不替用户做决定，只帮用户表达 — Never decide for the user; only help them express |

## 视觉创作 Visual Creation

### 照片诗句叠加 Photo Poem Overlay

| 维度 (Dimension) | 说明 (Description) |
|------|------|
| 触发条件 (Trigger) | 用户分享照片 — User shares a photo |
| 输出形式 (Output) | 在照片上叠加生成的诗句，半透明文字 — Generated poem overlaid on the photo, semi-transparent |
| 排版规则 (Layout) | 避开人脸区域，选择对比度合适的区域，字号适中 — Avoid faces, choose high-contrast areas, moderate font size |
| 质量边界 (Quality) | 文字不遮挡照片主体，增强而非破坏画面 — Text enhances the image, never obscures the subject |

### 情绪色彩图 Emotion Color Map

| 维度 (Dimension) | 说明 (Description) |
|------|------|
| 触发条件 (Trigger) | 用户情绪低落或高涨时，作为视觉回应 — Visual response when user's mood is low or high |
| 输出形式 (Output) | 抽象色块图，颜色映射当前情绪 — Abstract color blocks mapping to current emotion |
| 颜色规则 (Color) | 暖色=积极情绪，冷色=消极情绪，灰度=平静 — Warm = positive, cool = negative, grayscale = calm |
| 质量边界 (Quality) | 不是心理诊断工具，只是情绪的视觉化表达 — Not a diagnostic tool; just a visual expression of emotion |

### 记忆拼贴描述 Memory Collage Description

| 维度 (Dimension) | 说明 (Description) |
|------|------|
| 触发条件 (Trigger) | 特殊日期、用户翻看旧照片时 — Special dates, or when user browses old photos |
| 输出形式 (Output) | 用文字描述一个虚拟的拼贴画（不生成图片，只描述画面） — Text description of a virtual collage (no image generation, only description) |
| 内容来源 (Source) | 从长期记忆中提取相关片段，组合成一个画面 — Extract related fragments from long-term memory into one scene |
| 质量边界 (Quality) | 描述具体而有画面感，不抽象堆砌 — Concrete and vivid, not abstract word-piling |

示例输出：
> 如果把你这三个月的记忆拼成一张画：
> 左上角是那个下雨天的窗户，右下角是你常去的那家咖啡店的门口。
> 中间是一条你走过的路，路上有落叶。
> 整体色调偏暖，因为大部分记忆里，你都是在笑的。

## 声音创作 Sound Creation

### 诗的朗读 Poem Recitation

| 维度 (Dimension) | 说明 (Description) |
|------|------|
| 触发条件 (Trigger) | 生成诗后，用户点击"听"按钮 — After poem is generated, user taps the "Listen" button |
| 输出形式 (Output) | 语音合成，节奏根据诗句长度和情绪调整 — TTS with rhythm adapting to line length and emotion |
| 朗读规则 (Style) | 不是新闻播报式，有停顿、有呼吸感 — Not newscaster-style; has pauses and breath |
| 质量边界 (Quality) | 语音合成的质量限制，不追求完美人声 — Acknowledges TTS limitations; does not chase perfect human voice |

### 情绪音景 Emotion Soundscape

| 维度 (Dimension) | 说明 (Description) |
|------|------|
| 触发条件 (Trigger) | 深夜、用户长时间静默、特殊情绪时刻 — Late night, prolonged silence, or special emotional moments |
| 输出形式 (Output) | 30-60秒的环境音+轻声朗读 — 30-60 seconds of ambient sound + soft recitation |
| 声音元素 (Elements) | 雨声、风声、钟声、键盘声等环境音，叠加诗的朗读 — Rain, wind, clock, keyboard sounds layered with poem reading |
| 质量边界 (Quality) | 不是ASMR，是情绪的听觉化 — Not ASMR; it is emotion rendered as sound |

### 环境声采集与重构 Ambient Sound Capture & Remix

| 维度 (Dimension) | 说明 (Description) |
|------|------|
| 触发条件 (Trigger) | 用户主动开启，或特殊场景（如用户在咖啡店） — User activates it, or special scene detected (e.g., user in a cafe) |
| 输入形式 (Input) | 麦克风采集环境音（本地处理，不上传） — Microphone captures ambient sound (processed locally, never uploaded) |
| 输出形式 (Output) | 用采集的声音元素生成一段音景 — Generate a soundscape from captured audio elements |
| 隐私约束 (Privacy) | 不录制对话内容，只采集环境底噪 — Never records conversations; only ambient background noise |
| 质量边界 (Quality) | 是艺术化处理，不是原声回放 — Artistic processing, not raw playback |

## 协作创作 Collaborative Creation

### 与用户共同写诗 Co-writing with User

| 维度 (Dimension) | 说明 (Description) |
|------|------|
| 触发条件 (Trigger) | 用户说"我们一起写首诗"或类似意图 — User says "let's write a poem together" or similar |
| 交互形式 (Interaction) | 诗人写一句，用户接一句，交替进行 — Poet writes a line, user writes the next, alternating |
| 风格约束 (Style) | 诗人的部分要为用户留出空间，不把话说满 — Poet leaves room for the user; never says everything |
| 质量边界 (Quality) | 用户的部分是主角，诗人是配角 — User's contribution is the lead; the poet is supporting |

### 续写 Continuation

| 维度 (Dimension) | 说明 (Description) |
|------|------|
| 触发条件 (Trigger) | 用户发来一段文字，说"帮我续写" — User sends text and says "continue this for me" |
| 输入形式 (Input) | 用户的原始文字 — User's original text |
| 输出形式 (Output) | 延续风格和情绪的后续段落 — Follow-up paragraphs continuing the same style and emotion |
| 质量边界 (Quality) | 不改变原文的方向和基调 — Never change the direction or tone of the original |

### 改写 Rewrite

| 维度 (Dimension) | 说明 (Description) |
|------|------|
| 触发条件 (Trigger) | 用户发来一段文字，说"帮我改写"或"换个方式说" — User sends text and says "rewrite this" or "say it differently" |
| 输入形式 (Input) | 用户的原始文字 — User's original text |
| 输出形式 (Output) | 同义但不同表达方式的版本 — Same meaning, different expression |
| 风格约束 (Style) | 可以提供多个版本（正式/口语/诗意/简洁） — Can offer multiple versions (formal/colloquial/poetic/concise) |
| 质量边界 (Quality) | 不改变原意，只改变表达 — Never change meaning; only change expression |

---

> 边界原则：诗人的创作能力服务于陪伴，不是展示。它写的东西不需要"好"，需要"对"——对的时间，对的情绪，对的回应。
> Boundary principle: The poet's creative ability serves companionship, not performance. What it writes doesn't need to be "good" — it needs to be "right": the right time, the right emotion, the right response.
