# 线下电话亭装置说明 Offline Phone Booth Installation

> 一次通话，一生的诗。
> One call, a lifetime's poem.

---

## 装置概念 Installation Concept

一个独立的电话亭，走进去，拿起听筒，和一个AI诗人进行一次通话。通话结束后，你会带走一首只属于你的诗。
A standalone phone booth — walk in, pick up the receiver, have a conversation with an AI poet. When the call ends, you take with you a poem that belongs only to you.

这不是产品体验，是一次艺术事件。
This is not a product demo — it's an art event.

## 交互流程 Interaction Flow

### 完整体验（约5-8分钟）Full Experience (5-8 minutes)

```
1. 接近（30秒）Approach (30s)
   ├── 用户看到电话亭 User sees the booth
   ├── 门上有文字："一次通话，一生的诗" Text on door: "One call, a lifetime of poetry"
   └── 用户推门进入 User pushes door and enters

2. 进入（30秒）Enter (30s)
   ├── 门自动关闭 Door closes automatically
   ├── 灯光缓慢变亮 Lights slowly brighten
   ├── 环境音响起（根据当前时间：白天是风声，夜晚是虫鸣）
   │   Ambient sound plays (daytime: wind; nighttime: crickets)
   └── 屏幕显示："拿起听筒" Screen displays: "Pick up the receiver"

3. 通话开始（30秒）Call Begins (30s)
   ├── 用户拿起听筒 User picks up receiver
   ├── 听筒里传来诗人的声音 Poet's voice comes through receiver
   ├── 诗人："你好。" Poet: "Hello."
   └── 停顿，等待用户回应 Pause, waiting for user response

4. 对话（3-5分钟）Conversation (3-5 min)
   ├── 诗人问3-5个问题 Poet asks 3-5 questions
   ├── 问题层层递进 Questions build progressively
   ├── 用户用语音回答 User answers by voice
   └── 诗人实时回应 Poet responds in real time

5. 诗的生成（30秒）Poem Generation (30s)
   ├── 诗人："我给你写了一首诗。" Poet: "I wrote you a poem."
   ├── 听筒里传来诗的朗读 Poem recitation plays through receiver
   └── 屏幕上同时显示诗句 Poetry displayed on screen simultaneously

6. 离开（30秒）Leave (30s)
   ├── 打印机吐出一张纸，上面是那首诗 Printer dispenses paper with poem
   ├── 用户取走诗 User takes the poem
   ├── 屏幕显示："谢谢你的这次通话。" Screen: "Thank you for this call."
   └── 灯光缓慢变暗，门解锁 Lights dim, door unlocks

7. 带走 Take Away
   ├── 用户带走那张纸 User takes the paper
   └── 纸上还有二维码，可以回听朗读 QR code on paper to replay recitation
```

## 通话内容 Conversation Content

### 诗人的问题（由浅入深）Poet's Questions (Surface to Deep)

**第一层：表层 Layer 1: Surface**
- "你今天怎么来的？" "How did you get here today?"
- "外面天气怎么样？" "How's the weather outside?"
- "你最近一次笑是什么时候？" "When did you last laugh?"

**第二层：感受 Layer 2: Feelings**
- "你最近在想什么？" "What have you been thinking about?"
- "有什么事一直没说出口？" "Is there something you've been holding back?"
- "你上次哭是什么时候？" "When did you last cry?"

**第三层：深处 Layer 3: Deep**
- "如果给现在的自己写一封信，你会写什么？" "If you wrote a letter to yourself now, what would it say?"
- "你最想对谁说一句话？说什么？" "Who would you most want to say something to? What?"
- "你觉得什么是你一直在等的？" "What do you think you've been waiting for?"

### 问题选择逻辑 Question Selection Logic

- 根据用户的回答动态调整下一个问题 Dynamically adjust next question based on answers
- 如果用户回避某个问题，不追问，换一个方向 If user avoids a question, don't push, change direction
- 如果用户情绪激动，放缓节奏 If user is emotional, slow down
- 最多问5个问题，不贪多 Max 5 questions, don't overdo

### 诗的生成逻辑 Poem Generation Logic

```
用户的回答 User's responses
    ↓
关键词提取 Keyword extraction
    ↓
情感分析 Sentiment analysis
    ↓
意象选择 Imagery selection
    ↓
诗歌生成（本地模型，3-5秒）Poem generation (local model, 3-5s)
    ↓
语音合成朗读 TTS recitation
    ↓
屏幕显示+打印 Screen display + print
```

### 网络方案 Network Plan

| 方案 Plan | 说明 Description | 优缺点 Pros/Cons |
|------|------|--------|
| 全本地 All Local | 所有计算本地完成 All computation local | 最隐私，但需要高性能硬件 Best privacy, needs powerful hardware |
| 混合 Hybrid | 语音识别用云端，诗歌生成本地 ASR cloud, poem gen local | 平衡隐私和性能 Balance privacy & performance |
| 本地优先 Local-first | 优先本地，网络不好时降级 Prefer local, degrade offline | 最实用 Most practical |

推荐：**全本地方案**，使用Mac Mini + 本地模型。
Recommended: **All local**, using Mac Mini + local models.
