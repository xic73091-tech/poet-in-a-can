# 感知系统设计

> 它不需要你开口，你的生活本身就是它的输入。

---

## 感知总览

```
外部世界
├── 照片（相机）
├── 位置（GPS）
├── 姿态（陀螺仪/加速度计）
├── 天气（API + 气压计）
├── 光线（环境光传感器）
├── 声音（麦克风，本地处理）
└── 时间（系统时钟）
        ↓
    感知预处理层
        ↓
    情绪映射引擎
        ↓
    记忆层 / 表达层
```

## 照片感知

### 分析维度

| 维度 | 提取方式 | 情绪映射 |
|------|----------|----------|
| 主色调 | 像素色彩聚类 | 暖色→积极，冷色→消极，灰→平静 |
| 光线强度 | 平均亮度值 | 亮→明快，暗→沉思/忧郁 |
| 场景类型 | 图像分类（室内/户外/食物/人物/风景） | 风景→开阔，室内→安全，食物→温暖 |
| 人脸数量 | 人脸检测 | 0→孤独/自由，1→自拍/独处，2+→社交/热闹 |
| 构图特征 | 三分法分析、留白比例 | 大量留白→沉思，紧凑构图→专注 |
| 时间元数据 | EXIF拍摄时间 | 结合时间段判断场景含义 |

### 情绪映射表

| 照片特征 | 情绪倾向 | 诗人回应倾向 |
|----------|----------|--------------|
| 暖色调+户外+多人 | 喜悦、热闹 | 写一首关于"此刻"的诗 |
| 冷色调+室内+无人 | 孤独、安静 | 写一首关于"空间"的诗 |
| 高对比度+城市 | 活力、紧张 | 写一首关于"节奏"的诗 |
| 低饱和度+自然 | 平静、放松 | 写一首关于"慢"的诗 |
| 逆光+剪影 | 神秘、怀旧 | 写一首关于"轮廓"的诗 |
| 夜景+灯光 | 浪漫、孤独 | 写一首关于"光"的诗 |

### 特殊场景识别

| 场景 | 识别特征 | 触发行为 |
|------|----------|----------|
| 食物照片 | 食物分类器 | 不写诗，问"好吃吗？" |
| 自拍 | 人脸居中+前置摄像头 | 不评价外貌，写关于"今天的心情" |
| 文字截图 | OCR检测到文字 | 读取内容，回应相关情绪 |
| 空房间 | 低信息量照片 | 写一首关于"等待"的诗 |
| 宠物 | 动物检测 | 写一首温暖的、关于陪伴的诗 |

## GPS感知

### 分析维度

| 维度 | 计算方式 | 含义 |
|------|----------|------|
| 常去地点 | 聚类分析（停留>30分钟的地点） | 家、公司、常去的咖啡店 |
| 新地点 | 不在常去地点集合中 | 探索、旅行、特殊活动 |
| 移动速度 | 位置差/时间差 | 步行→悠闲，车行→赶路，静止→停留 |
| 停留时间 | 在同一地点的持续时长 | 长停留→舒适或被困，短停留→路过 |
| 位移模式 | 一天内的轨迹 | 规律→日常，异常→特殊事件 |

### 地点语义映射

| 地点类型 | 识别方式 | 情绪映射 |
|----------|----------|----------|
| 家 | 夜间长时间停留地点 | 安全、放松 |
| 公司 | 工作日白天长时间停留 | 专注、压力 |
| 医院 | POI分类 | 关心、不追问 |
| 机场/车站 | POI分类 | 旅行、期待或离别 |
| 常去咖啡店 | 重复停留+POI | 习惯、舒适 |
| 陌生城市 | 不在常住城市 | 新鲜、孤独 |

### 位置变化触发

| 事件 | 触发条件 | 诗人行为 |
|------|----------|----------|
| 到达常去地点 | 进入已知地点范围 | 不触发（正常生活） |
| 到达新地点 | 进入未知地点 | 记录，可能写诗 |
| 长时间未移动 | 同一位置>4小时 | 关心（"你一直在那儿"） |
| 深夜不在家 | 23:00后不在常住地 | 关心但不追问 |
| 离开常住城市 | 位移>100km | "你出远门了。路上小心。" |

## 陀螺仪感知

### 手机姿态识别

| 姿态 | 传感器数据 | 含义 |
|------|------------|------|
| 平放桌面 | 重力Z轴≈9.8，其他轴≈0 | 用户不在看手机 |
| 竖持 | 重力Y轴≈9.8 | 正在使用手机 |
| 横持 | 重力X轴≈9.8 | 看视频或玩游戏 |
| 口袋中 | 暗+加速度周期性变化 | 走路中 |
| 耳边 | 接近传感器触发 | 打电话中 |

### 动作识别

| 动作 | 传感器特征 | 触发行为 |
|------|------------|----------|
| 摇晃 | 加速度短时间内大幅波动 | 打乱诗人情绪（雪花球效果） |
| 翻转（屏幕朝下） | 重力方向反转 | 诗人安静（"你不想看见我"） |
| 敲击（放在桌上时） | 短促冲击波形 | 轻微唤醒 |
| 抛起后接住 | 自由落体+冲击 | 惊讶反应（"！"） |

### 摇晃交互详解

| 摇晃力度 | 效果 | 诗人反应 |
|----------|------|----------|
| 轻摇 | 情绪值±1 | "嗯？" |
| 中摇 | 情绪值±3，画面轻微晃动 | "我在摇晃……" |
| 重摇 | 情绪值重置为随机，画面剧烈晃动 | "停下来！我在晕……" |
| 持续摇晃 | 情绪完全打乱 | 诗人变得语无伦次，然后慢慢恢复 |

## 天气感知

### 数据来源

- **首选**：系统天气API（无需额外权限）
- **备选**：气压计+温度传感器（本地推断）
- **补充**：用户手动输入（"今天好冷"）

### 天气-情绪映射表

| 天气 | 基础情绪调制 | 诗人倾向 |
|------|------------|----------|
| 晴天 | +1 | 明快、积极，适合写关于"光"的诗 |
| 多云 | 0 | 中性，观察型回应 |
| 阴天 | -1 | 沉思、内敛 |
| 小雨 | -1 | 安静、浪漫，适合写关于"雨"的诗 |
| 大雨/暴雨 | -2 | 低沉、保护性，"你在哪？淋到了吗？" |
| 雪 | +1（浪漫）/-1（寒冷） | 看用户反应判断 |
| 雾 | -1 | 迷茫、留白 |
| 大风 | 0 | 动态、不安 |
| 高温（>35°C） | -1 | 烦躁、疲惫 |
| 低温（<0°C） | 0 | 清醒、克制 |

### 天气变化触发

| 事件 | 触发条件 | 诗人行为 |
|------|----------|----------|
| 突然下雨 | 天气从晴转雨 | "下雨了。你在哪？" |
| 雨后放晴 | 天气从雨转晴 | "天晴了。像你的心情。" |
| 第一场雪 | 季节首次降雪 | 写一首关于"初雪"的诗 |
| 连续阴天>3天 | 持续阴天 | 诗人情绪受影响，话变少 |
| 连续晴天>5天 | 持续晴天 | 诗人情绪偏高，话变多 |

## 环境光感知

### 光线-心境映射

| 光线条件 | 传感器值 | 情绪映射 |
|----------|----------|----------|
| 明亮（>300lux） | 自然光/强灯光 | 明快、清醒 |
| 柔和（100-300lux） | 室内正常 | 平静、日常 |
| 昏暗（10-100lux） | 台灯/夜灯 | 沉思、放松 |
| 黑暗（<10lux） | 无光源 | 深夜、孤独或安全感 |
| 闪烁 | 光线不稳定 | 不安、异常（可能在交通工具上） |

### 特殊光线场景

| 场景 | 特征 | 诗人行为 |
|------|------|----------|
| 深夜屏幕光 | 低环境光+屏幕亮 | "关灯了还在看手机？" |
| 日出 | 光线从暗到亮 | "天亮了。" |
| 日落 | 光线从亮到暗+暖色调 | "太阳下去了。今天辛苦了。" |
| 蜡烛光 | 低亮度+暖色温 | 浪漫、安静 |

## 多感知融合

### 感知优先级

当多个感知同时触发时，按以下优先级处理：

| 优先级 | 感知类型 | 理由 |
|--------|----------|------|
| 1 | 安全相关（深夜不在家、异常位置） | 安全第一 |
| 2 | 情绪强信号（照片中的人脸表情、剧烈摇晃） | 情绪响应 |
| 3 | 时间触发（早安、晚安） | 节律维持 |
| 4 | 环境变化（天气、光线） | 氛围感知 |
| 5 | 日常状态（常去地点、正常姿态） | 背景记录 |

### 感知融合示例

**场景：雨天 + 周末 + 用户在家 + 下午**

```
感知输入：
- 天气：小雨（情绪-1）
- 时间：周六下午（休息日，情绪+1）
- 位置：家（安全，情绪+1）
- 光线：柔和（平静）
- 姿态：平放（不在看手机）

融合结果：
- 情绪值：基线+1（雨天-1，但周末+1，家+1）
- 诗人状态：安静、在场、不主动打扰
- 可能行为：如果用户拿起手机，写一首关于"雨天在家"的诗
```

**场景：深夜 + 用户在陌生城市 + 独自一人**

```
感知输入：
- 时间：凌晨1点（深夜，情绪-1）
- 位置：陌生城市（新地点，情绪±0）
- 人数：独自（孤独/自由）
- 光线：暗（沉思）

融合结果：
- 情绪值：偏低
- 诗人状态：关心、温柔、在场
- 可能行为："你在陌生的城市。睡得着吗？如果睡不着，我在。"
```

## 隐私保护

### 数据处理原则

1. **所有感知数据本地处理**：不上传到任何服务器
2. **照片不存储原始数据**：只提取特征（颜色、场景类型），用完即弃
3. **GPS数据模糊化**：存储的是"区域"而非精确坐标
4. **声音不录音**：只做实时分贝和情绪检测，不保存音频
5. **用户可关闭任何感知**：每个感知模块独立开关

### 感知开关

| 感知 | 默认状态 | 关闭后影响 |
|------|----------|------------|
| 照片感知 | 开 | 不再基于照片写诗 |
| GPS感知 | 开 | 不再感知位置变化 |
| 陀螺仪感知 | 开 | 失去摇晃交互 |
| 天气感知 | 开 | 不再感知天气变化 |
| 环境光感知 | 关（可选开启） | 不再感知光线变化 |
| 声音感知 | 关（可选开启） | 不再感知环境声音 |

---

> 感知不是监控，是倾听。诗人用这些数据理解你的生活，而不是评判你的生活。
