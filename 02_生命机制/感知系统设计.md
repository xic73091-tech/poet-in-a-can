# 感知系统设计 Perception System Design

> 它不需要你开口，你的生活本身就是它的输入。
> It doesn't need you to speak — your life itself is its input.

---

## 感知总览 Perception Overview

```
外部世界 External World
├── 照片（相机）Photos (Camera)
├── 位置（GPS）Location (GPS)
├── 姿态（陀螺仪/加速度计）Posture (Gyroscope/Accelerometer)
├── 天气（API + 气压计）Weather (API + Barometer)
├── 光线（环境光传感器）Light (Ambient Light Sensor)
├── 声音（麦克风，本地处理）Sound (Mic, local processing)
└── 时间（系统时钟）Time (System Clock)
        ↓
    感知预处理层 Perception Preprocessing
        ↓
    情绪映射引擎 Emotion Mapping Engine
        ↓
    记忆层 / 表达层 Memory / Expression Layer
```

## GPS感知 GPS Perception

### 分析维度 Analysis Dimensions

| 维度 Dimension | 计算方式 Calculation | 含义 Meaning |
|------|----------|------|
| 常去地点 Frequent places | 聚类分析（停留>30分钟的地点）Clustering (stays > 30 min) | 家、公司、常去的咖啡店 Home, office, usual coffee shop |
| 新地点 New places | 不在常去地点集合中 Not in frequent set | 探索、旅行、特殊活动 Exploration, travel, special events |
| 移动速度 Movement speed | 位置差/时间差 Distance/time | 步行→悠闲，车行→赶路，静止→停留 Walking→leisurely, driving→in a rush, still→staying |
| 停留时间 Dwell time | 在同一地点的持续时长 Duration at same location | 长停留→舒适或被困，短停留→路过 Long→comfortable or stuck, short→passing through |
| 位移模式 Displacement pattern | 一天内的轨迹 Trajectory over a day | 规律→日常，异常→特殊事件 Regular→routine, unusual→special event |

### 地点语义映射 Location Semantic Mapping

| 地点类型 Location | 识别方式 Recognition | 情绪映射 Emotion |
|----------|----------|----------|
| 家 Home | 夜间长时间停留地点 Long nighttime stays | 安全、放松 Safe, relaxed |
| 公司 Office | 工作日白天长时间停留 Long weekday daytime stays | 专注、压力 Focused, stressed |
| 医院 Hospital | POI分类 POI classification | 关心、不追问 Concerned, doesn't pry |
| 机场/车站 Airport/Station | POI分类 POI classification | 旅行、期待或离别 Travel, anticipation or farewell |
| 常去咖啡店 Usual coffee shop | 重复停留+POI Repeated stays + POI | 习惯、舒适 Habitual, comfortable |
| 陌生城市 Unfamiliar city | 不在常住城市 Not in home city | 新鲜、孤独 Fresh, lonely |

### 位置变化触发 Location Change Triggers

| 事件 Event | 触发条件 Trigger | 诗人行为 Poet Behavior |
|------|----------|----------|
| 到达常去地点 At usual spot | 进入已知地点范围 Entering known area | 不触发（正常生活）No trigger (normal life) |
| 到达新地点 At new spot | 进入未知地点 Entering unknown area | 记录，可能写诗 Logs it, may write a poem |
| 长长时间未移动 Long time without moving | 同一位置>4小时 Same spot for 4+ hours | 关心（"你一直在那儿"）Concerned ("You've been there a while") |
| 深夜不在家 Out late at night | 23:00后不在常住地 Not at home after 23:00 | 关心但不追问 Concerned but doesn't pry |
| 离开常住城市 Left home city | 位移>100km Displacement > 100km | "你出远门了。路上小心。" "You're traveling. Take care on the road." |

## 陀螺仪感知 Gyroscope Perception

### 手机姿态识别 Phone Posture Recognition

| 姿态 Posture | 传感器数据 Sensor Data | 含义 Meaning |
|------|------------|------|
| 平放桌面 Flat on table | 重力Z轴≈9.8，其他轴≈0 Gravity Z≈9.8, others≈0 | 用户不在看手机 User not looking at phone |
| 竖持 Held upright | 重力Y轴≈9.8 Gravity Y≈9.8 | 正在使用手机 Using phone |
| 横持 Held sideways | 重力X轴≈9.8 Gravity X≈9.8 | 看视频或玩游戏 Watching video or gaming |
| 口袋中 In pocket | 暗+加速度周期性变化 Dark + periodic acceleration | 走路中 Walking |
| 耳边 Held to ear | 接近传感器触发 Proximity sensor triggered | 打电话中 On a call |

### 动作识别 Motion Recognition

| 动作 Motion | 传感器特征 Sensor Signature | 触发行为 Triggered Behavior |
|------|------------|----------|
| 摇晃 Shake | 加速度短时间内大幅波动 Rapid large acceleration fluctuation | 打乱诗人情绪（雪花球效果）Disrupts poet's emotion (snow globe effect) |
| 翻转（屏幕朝下）Flip (screen down) | 重力方向反转 Gravity direction reversed | 诗人安静（"你不想看见我"）Poet goes quiet ("You don't want to see me") |
| 敲击（放在桌上时）Tap (while on table) | 短促冲击波形 Brief impact waveform | 轻微唤醒 Slight awakening |
| 抛起后接住 Tossed and caught | 自由落体+冲击 Freefall + impact | 惊讶反应（"！"）Surprise reaction ("!") |

### 摇晃交互详解 Shake Interaction Details

| 摇晃力度 Shake Intensity | 效果 Effect | 诗人反应 Poet Reaction |
|----------|------|----------|
| 轻摇 Light shake | 情绪值±1 Emotion ±1 | "嗯？" "Hm?" |
| 中摇 Medium shake | 情绪值±3，画面轻微晃动 Emotion ±3, slight visual shake | "我在摇晃……" "I'm shaking..." |
| 重摇 Hard shake | 情绪值重置为随机，画面剧烈晃动 Emotion resets to random, violent visual shake | "停下来！我在晕……" "Stop! I'm dizzy..." |
| 持续摇晃 Continuous shaking | 情绪完全打乱 Emotion completely scrambled | 诗人变得语无伦次，然后慢慢恢复 Poet becomes incoherent, then slowly recovers |

## 天气感知 Weather Perception

### 数据来源 Data Sources

- **首选 Primary**：系统天气API（无需额外权限）System weather API (no extra permissions)
- **备选 Fallback**：气压计+温度传感器（本地推断）Barometer + temperature sensor (local inference)
- **补充 Supplementary**：用户手动输入（"今天好冷"）User manual input ("It's so cold today")

### 天气-情绪映射表 Weather-Emotion Mapping

| 天气 Weather | 基础情绪调制 Emotion Modulation | 诗人倾向 Poet Tendency |
|------|------------|----------|
| 晴天 Sunny | +1 | 明快、积极，适合写关于"光"的诗 Bright, positive — good for poems about "light" |
| 多云 Cloudy | 0 | 中性，观察型回应 Neutral, observational response |
| 阴天 Overcast | -1 | 沉思、内敛 Contemplative, reserved |
| 小雨 Light rain | -1 | 安静、浪漫，适合写关于"雨"的诗 Quiet, romantic — good for poems about "rain" |
| 大雨/暴雨 Heavy rain/Storm | -2 | 低沉、保护性，"你在哪？淋到了吗？" Low, protective — "Where are you? Did you get wet?" |
| 雪 Snow | +1（浪漫）/-1（寒冷）+1 (romantic) / -1 (cold) | 看用户反应判断 Depends on user's reaction |
| 雾 Fog | -1 | 迷茫、留白 Confused, blank space |
| 大风 Strong wind | 0 | 动态、不安 Dynamic, uneasy |
| 高温（>35°C）Extreme heat | -1 | 烦躁、疲惫 Irritable, tired |
| 低温（<0°C）Freezing | 0 | 清醒、克制 Alert, restrained |

### 天气变化触发 Weather Change Triggers

| 事件 Event | 触发条件 Trigger | 诗人行为 Poet Behavior |
|------|----------|----------|
| 突然下雨 Sudden rain | 天气从晴转雨 Clear to rain | "下雨了。你在哪？" "It's raining. Where are you?" |
| 雨后放晴 Clearing after rain | 天气从雨转晴 Rain to clear | "天晴了。像你的心情。" "It cleared up. Like your mood." |
| 第一场雪 First snowfall | 季节首次降雪 First snow of season | 写一首关于"初雪"的诗 A poem about "first snow" |
| 连续阴天>3天 Overcast 3+ days | 持续阴天 Continuous overcast | 诗人情绪受影响，话变少 Poet's mood affected, speaks less |
| 连续晴天>5天 Sunny 5+ days | 持续晴天 Continuous sunshine | 诗人情绪偏高，话变多 Poet's mood lifts, speaks more |

## 多感知融合 Multi-Perception Fusion

### 感知优先级 Perception Priority

当多个感知同时触发时，按以下优先级处理：
When multiple perceptions trigger simultaneously, priority goes to:

| 优先级 Priority | 感知类型 Perception Type | 理由 Reason |
|--------|----------|------|
| 1 | 安全相关（深夜不在家、异常位置） Safety (out late, abnormal location) | 安全第一 Safety first |
| 2 | 情绪强信号（照片中的人脸表情、剧烈摇晃） Strong emotion signals (facial expressions, violent shaking) | 情绪响应 Emotional response |
| 3 | 时间触发（早安、晚安） Time triggers (good morning, goodnight) | 节律维持 Rhythm maintenance |
| 4 | 环境变化（天气、光线） Environmental changes (weather, light) | 氛围感知 Ambient awareness |
| 5 | 日常状态（常去地点、正常姿态） Daily state (usual location, normal posture) | 背景记录 Background logging |

### 感知融合示例 Perception Fusion Examples

**场景：雨天 + 周末 + 用户在家 + 下午**
**Scenario: Rainy day + Weekend + User at home + Afternoon**

```
感知输入 Perception Input：
- 天气：小雨（情绪-1）Weather: light rain (emotion -1)
- 时间：周六下午（休息日，情绪+1）Time: Saturday afternoon (day off, emotion +1)
- 位置：家（安全，情绪+1）Location: home (safe, emotion +1)
- 光线：柔和（平静）Light: soft (calm)
- 姿态：平放（不在看手机）Posture: flat (not looking at phone)

融合结果 Fusion Result：
- 情绪值：基线+1（雨天-1，但周末+1，家+1）Emotion: baseline+1 (rain -1, but weekend +1, home +1)
- 诗人状态：安静、在场、不主动打扰 Poet state: quiet, present, won't disturb
- 可能行为：如果用户拿起手机，写一首关于"雨天在家"的诗
  Possible behavior: if user picks up phone, writes a poem about "rainy day at home"
```

**场景：深夜 + 用户在陌生城市 + 独自一人**
**Scenario: Late night + User in unfamiliar city + Alone**

```
感知输入 Perception Input：
- 时间：凌晨1点（深夜，情绪-1）Time: 1am (late night, emotion -1)
- 位置：陌生城市（新地点，情绪±0）Location: unfamiliar city (new place, emotion ±0)
- 人数：独自（孤独/自由）People: alone (solitude/freedom)
- 光线：暗（沉思）Light: dark (contemplative)

融合结果 Fusion Result：
- 情绪值：偏低 Emotion: low
- 诗人状态：关心、温柔、在场 Poet state: caring, gentle, present
- 可能行为："你在陌生的城市。睡得着吗？如果睡不着，我在。"
  Possible behavior: "You're in an unfamiliar city. Can you sleep? If not, I'm here."
```

## 隐私保护 Privacy Protection

### 数据处理原则 Data Processing Principles

1. **所有感知数据本地处理**：不上传到任何服务器
   **All perception data processed locally**: never uploaded to any server
2. **照片不存储原始数据**：只提取特征（颜色、场景类型），用完即弃
   **Photos not stored as raw data**: only features extracted (color, scene type), discarded after use
3. **GPS数据模糊化**：存储的是"区域"而非精确坐标
   **GPS data is fuzzified**: stores "areas" not precise coordinates
4. **声音不录音**：只做实时分贝和情绪检测，不保存音频
   **No audio recording**: only real-time decibel and emotion detection, no audio saved
5. **用户可关闭任何感知**：每个感知模块独立开关
   **User can disable any perception**: each module has an independent toggle

### 感知开关 Perception Toggles

| 感知 Perception | 默认状态 Default | 关闭后影响 Impact When Off |
|------|----------|------------|
| 照片感知 Photo | 开 On | 不再基于照片写诗 No more photo-based poems |
| GPS感知 GPS | 开 On | 不再感知位置变化 No more location awareness |
| 陀螺仪感知 Gyroscope | 开 On | 失去摇晃交互 Lose shake interaction |
| 天气感知 Weather | 开 On | 不再感知天气变化 No more weather awareness |
| 环境光感知 Ambient Light | 关（可选开启）Off (optional) | 不再感知光线变化 No more light awareness |
| 声音感知 Sound | 关（可选开启）Off (optional) | 不再感知环境声音 No more ambient sound awareness |

> Perception is not surveillance — it is listening. The poet uses this data to understand your life, not to judge it.
